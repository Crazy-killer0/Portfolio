FCSC 2024 - Silence - Forensic

2024-04-14
2 minute read
FCSC2024
CTF , Forensic

Introduction

You’re Bob. An old friend sent you a flag for a future FCSC a few years ago, before you lost touch. She reminds you that she is now part of the FCSC organisation. She has been able to set up this flag and offers to play it for you.

The problem is that the flag has been left in an SMS backup in your SMS application of the time, Silence. You’ve completely forgotten the code you used, you only know that it was a 5-digit decimal code. Can you find this flag and show Alice that you haven’t forgotten her?

Any resemblance to facts or characters that exist or have existed is purely coincidental. No AI has been mistreated in the making of this challenge.

Unfortunately, you don’t need to reverse engineer the Silence application to find flaws, as someone has already taken Silence’s Java code and used it in a ‘blackbox’ to create a decryption tool.

You follow the procedures for installing the tool, and when it comes to decrypting it, you’re asked for a password.

We know that the password is only 5 digits long, so we can try to bruteforce it.

I modified the scripts run-jar.sh and run-all.sh to take a password from the command-line rather than stdin.

run-jar.sh

#!/bin/sh -e

cd "$(dirname "$0")"

if [ $# -ne 2 ]; then
        echo "usage: $0 SILENCE_EXPORT_DIR_PATH PASSWORD" >&2
        exit 64
fi

SILENCE_EXPORT_DIR_PATH=$1
PASSWORD=$2

if [ -z "$BCPROV_JAR" ]
then
	BCPROV_JAR=$(ls bcprov*.jar | sed 1q)
fi

echo "$PASSWORD" | java -cp "$BCPROV_JAR:build/breakthesilence.jar" re.indigo.breakthesilence.MasterSecretUtil "$SILENCE_EXPORT_DIR_PATH/shared_prefs/SecureSMS-Preferences.xml"

`run-all.sh``

#!/bin/sh -e

cd "$(dirname "$0")"

props=$(mktemp silence-props.XXXXXX)
trap "rm $props" EXIT

./run-jar.sh "$1" "$3" > "$props"
./breakthesilence_to_json.py "$1" "$2" < "$props"

We then write a bash script that uses parallel to parallelise the tests.

break.sh

#!/bin/bash
RUN_ALL_SCRIPT="./run-all.sh"

SILENCEEXPORT_DIR_PATH="../SilenceExport"

OUTPUT_FILE="silence-backup.json"

try_password() {
    local password=$1
    echo "Trying password: $password"
    if $RUN_ALL_SCRIPT $SILENCEEXPORT_DIR_PATH $OUTPUT_FILE $password; then
        echo "Password found: $password"
        kill 0
    fi
}

export -f try_password
export RUN_ALL_SCRIPT
export SILENCEEXPORT_DIR_PATH
export OUTPUT_FILE

parallel -j24 try_password ::: {0..99999}
